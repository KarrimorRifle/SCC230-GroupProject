# Actors
## Users
- Sign Up
  - Requires email validation
- Login
  - Requires 2FA
- Logout
- Delete Account
- Join Hub
- Edit Settings
- reset password
- Create Hub

### Hub  
- Create Schedule
- Edit schedule (must be on draft mode)
- Generate Schedule
- request activation

#### Hub Admin
- Invite user
- Remove User
- Add schedule
- Draft Schedule
- Undraft Schedule
- Delete Schedule
- Opt out of AI usage
- Edit Schedule (Must not be active)
- Accept/deny user (Hub inviting)

##### Hub Owner
- Edit Hub Settings
- Add devices
- Remove devices
- Activate Schedule
- Deactivate schedule
